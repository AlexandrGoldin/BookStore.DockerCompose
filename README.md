# BookStore.Server
C# Backend of the Client-Server PC pet-project of the online sopping "BookStore". 

### Developed in Visual Studio 2019

### Stack:
* MS SQL Server
* TSQL
* ASP.NET Core 3.1
* Web API
* Entity Framework Core
* N-tier architecture
* CORS
* Swashbuckle
* Data Transfer Object
* AutoMapper
* MemoryCache
* xUnit
